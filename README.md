## Initialize and install required modules

```bash
npm init -y  
npm install
cd frontend
npm install react-hot-toast react-router-dom axios zustand lucide-react
```


## Setup .env file

```env
PORT=3000  
PGUSER=...  
PGPASSWORD=...  
PGHOST=...  
PGDATABASE=...
  
ARCJET_KEY=...  
ARCJET_ENV=development
```

## Run the API

```bash
npm run dev
```

## Run Frontend

```bash
cd frontend
npm run dev
```
